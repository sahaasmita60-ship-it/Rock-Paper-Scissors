# Rock, Paper, Scissors Game in Python

A simple **command-line Rock, Paper, Scissors game** implemented in Python. Play against the computer and see if you can beat it!

## About

Rock, Paper, Scissors is a classic hand game played between two players, where each player simultaneously chooses one of three options: **rock**, **paper**, or **scissors**. The rules are simple:

- Rock beats Scissors  
- Scissors beats Paper  
- Paper beats Rock  
- Identical choices result in a tie  

This Python script simulates the game against the computer using random choices.

## Features

- Play against the computer.  
- Input your choice via the command line.  
- The computer's choice is randomly generated.  
- Announces the winner or a tie.
